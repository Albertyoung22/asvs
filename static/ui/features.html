🎙️ OmniSignal 智控系統

整合 UDP 語音接收、多媒體播放、智能排程、地震預警的全方位場域廣播解決方案。 支援桌面 UI 與 Web 介面雙重控制，讓播報工作更智能、更便捷。

📚 查看使用說明
🔊

多層 TTS 語音引擎

智能備援機制，確保語音播報永不中斷：

  • Edge TTS - 高品質雲端語音（優先）
  • Piper TTS - 離線本地語音引擎
  • gTTS - Google 文字轉語音
  • SAPI5 - Windows 內建語音（備援）
  • 支援中文、英文、日文、韓文、越南語、印尼語、台語
🎵

多媒體播放系統

支援多種音訊格式與來源：

  • MP3 檔案上傳與管理
  • YouTube 影片下載轉 MP3
  • 全螢幕影片播放（mpv）
  • 播放控制（暫停/繼續/停止）
  • 音量調整與靜音功能
📅

智能排程系統

自動化排程，解放人力：

  • 自動播放排程（開場、結束、提醒）
  • 自訂排程（一次性/重複性）
  • 支援多種動作（TTS、MP3、YouTube）
  • 視覺化排程管理介面
  • 即時預覽下一次觸發時間
🌋

地震預警整合

即時接收中央氣象署地震資訊：

  • 自動偵測伺服器位置
  • 強度門檻過濾
  • 自動播放警報音效
  • TTS 廣播地震詳情
  • WebSocket 即時推送
🎤

即時錄音廣播

遠端麥克風功能，隨時隨地廣播：

  • 按住說話（Hold to Talk）
  • 錄音後預覽再傳送
  • 支援桌面與行動裝置
  • 自動音訊壓縮
  • 即時播放無延遲
🌐

多語言翻譯

內建翻譯引擎，打破語言障礙：

  • 支援 Google 翻譯 API
  • 台語 ↔ 中文 雙向翻譯
  • 自動偵測來源語言
  • 一鍵翻譯並播報
  • 支援 7+ 種語言
🤖

AI 廣播稿生成

整合 Ollama 本地 LLM：

  • 輸入關鍵字自動生成廣播稿
  • 語氣親切友善
  • 適合多樣化場域
  • 支援繁體中文
  • 離線運作保護隱私
📱

多平台支援

隨時隨地控制廣播系統：

  • 桌面 UI（Tkinter）
  • Web 介面（響應式設計）
  • 行動裝置優化
  • PWA 支援（可安裝）
  • ngrok 公網穿透（可選）
👥

接收端管理

完整的設備控制系統：

  • 自動發現接收端設備
  • 遠端開關機（WOL）
  • 分組管理
  • 即時狀態監控
  • WebRTC 視訊通話
🔐

安全認證系統

保護系統安全：

  • bcrypt 密碼雜湊
  • 角色權限管理（管理員/教師）
  • Session 會話管理
  • 登入/登出功能
  • API 端點保護
📣

分區廣播控制

4 路繼電器分區控制：

  • 獨立控制 4 個區域
  • 串列埠（COM Port）通訊
  • 即時狀態顯示
  • Web 介面控制
  • 支援自動化排程
🌤️

天氣播報整合

自動獲取即時天氣資訊：

  • IP 位置自動偵測
  • Open-Meteo API 整合
  • 溫度、濕度、天氣狀況
  • 空氣品質 AQI 指數
  • 一鍵插入氣象文字
